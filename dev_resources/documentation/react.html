Installation Guide

  • There are system requirements in order to install, Make sure that you are running the node.js and npm in your computer.You have to maintain this version (Node js v18.16.0 to v20.9.0)
    • Node.js and NPM :
  • After unzipping Template Files, you will find our product DreamsLMS->Reactjs(Source code), Documentation in the following Main Files folder.
  • Mandatory are npm and node are needed to run the reactjs project. If NodeJS is not installed then download it from https://nodejs.org/. NPM comes bundled with Node.js
  • To create a new application,

    • npm install -g create-react-app

  • To install the packages,run the command:

    • npm install

  • For running a project,run the command:

    • npm start

  • To build a file, run the command
  • If you want to run the project and make the build in the production mode then run the following command in the root directory, otherwise the project will continue to run in the development mode.

    • npm run build
